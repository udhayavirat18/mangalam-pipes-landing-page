# Frontend Developer Assignment - Mangalam Pipes

A pixel-perfect, fully responsive landing page built from scratch to match provided Figma specifications. 

👤 **Developer:** Udhayakumar  
🔗 **Live Demo:** https://udhayavirat18.github.io/mangalam-pipes-landing-page/

## 🚀 Key Features
* **Pixel-Perfect UI:** Meticulously translated from Figma designs with exact spacing, typography, and color matching.
* **Fluid Responsiveness:** Custom CSS Media Queries ensuring a seamless layout across Desktop (1440px/1200px), Tablet (1080px/800px), and Mobile (300px+) breakpoints.
* **Interactive Image Carousel:** Vanilla JavaScript implementation featuring thumbnail swapping and a custom coordinate-based "Hover-to-Zoom" functionality.
* **Dynamic Modals:** Fully functional popups for "Catalogue Download" and "Request Call Back" features with form validation UI.
* **Sticky Navigation:** JavaScript-triggered sticky header that appears after scrolling past the hero fold.

## 🛠️ Tech Stack
* **HTML5:** Semantic, accessible structure.
* **CSS3:** Flexbox, CSS Grid, custom properties, and fluid media queries.
* **JavaScript (Vanilla):** DOM manipulation and event handling without the overhead of external libraries or frameworks.

## 💻 Local Setup
Since this project uses entirely vanilla web technologies with zero dependencies, you can run it instantly:
1. Clone the repository.
2. Open `index.html` directly in any modern web browser.
3. No `npm install` or build steps required!
